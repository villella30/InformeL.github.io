Reloj ESP8266

Este es un reloj que se puede conectar a internet para que así te pueda dar la hora, el clima, la hora y que lo puedas usar como cronómetro y alarma. El microcontrolador que usamos es un ESP8266 ya que tiene conexión a internet y se puede programar por Arduino IDE. Para que esté pueda subir el código usamos un programador y le soldamos dos botones para bootear y resetear. Para ver todo esto usamos una pantalla OLED de 0,96 pulgadas y 128×64 píxeles. Como no es táctil, también soldamos 3 botones para poder navegar por el reloj. Como alimentación de todo esto usamos una batería de litio 2032 de 3V o mejor conocidas conocidas como “las pastillitas” para las calculadoras. Esta va en un portapilas de mother de pc que puede ir soldada al pcb o ir “suelta” según el usuario prefiera.
